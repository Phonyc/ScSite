Tangente & nombre dérivé

Tangente

Prenons une courbe représentative d'une fonction.

Prenons 2 Points sur cette courbe.

Traçons la droite passant par ces 2 points

Diminuez maintenant la distance (à droite) entre les 2 points jusqu'à ce qu'elle soit nulle.

100

On obtient alors la tangente à la courbe au point.

Le coefficient directeur de cette droite (pente) est le nombre dérivé de la fonction en ce point.

Nombre dérivé

$$ \text{Soit une fonction f définie sur un intervalle } I \text{ contenant } a\newline\text{ }\newline \text{Si } f \text{ est dérivable en } a \text{ il existe un réel tel que: }\newline\text{ }\newline f'(a)=\lim_{h\to 0}(\frac{f(a+h)-f(a)}{h})\newline\text{ }\newline \text{Soit la courbe } \mathscr{C}:y=f(x) \newline \text{Soit } A \text{ le point d'abscisse } a \text{ de la courbe } C_f \newline\text{ }\newline \text{La tangente à la courbe en } a \text{ est la droite passant par } A \text{ et de coefficient directeur } f'(a)\newline\text{ }\newline \text { En faisant tendre } h \text{ vers 0, la droite ne coupe plus la courbe qu'à un point}\newline $$
